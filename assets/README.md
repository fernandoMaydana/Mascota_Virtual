# Carpeta de Recursos Visuales (`assets/`) 🌸

¡Aquí puedes colocar los GIFs o imágenes animadas de Pompompurin!

### Archivo GIF recomendado:
- **`purin_saludando.gif`**: Coloca aquí tu GIF animado favorito de Pompompurin saludando o moviéndose.

> 💡 **Nota**: La aplicación detecta automáticamente la presencia de `assets/purin_saludando.gif`. Si la imagen no está presente o falla al cargar, la app activará de forma transparente la **mascota vectorial SVG animada integrada**, garantizando que nunca se rompa la interfaz.
